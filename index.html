用Hexo搭建个人博客

Posted on 2019-07-09

Hexo 搭建个人博客

安装node.js

windows下安装node.js，官方网站下载，并无脑next

安装git

windows下安装git，官方网站下载，一样无脑next
安装完成后，右键鼠标会出现下图所示内容：
将git配置好之后（这里默认读者已经配置好）

安装Hexo

  • 桌面右键鼠标，点击Git Bash Here，输入npm命令即可安装

  • 初始化配置

    根据喜好创建文件夹（例如hexo），并在文件夹下右键 git base here

本地查看效果

打开浏览器，在地址栏输入 localhost:4000 查看效果

将博客部署到github上

  • 登录自己的github，并打开刚才创建的repository
  • 复制ssh地址
  • 打开刚开始创建的文件夹hexo，编辑_config.yml文件
  • 做如下修改：
  • 在hexo文件夹下执行hexo g -d，登录我们自己的部署在Github Pages服务器上的博客了
  • 假如这时候，报错 ERROR Deployer not found: git，那么就是你的deployer没有安装成功，你需要执行如下命令再安装一次：npm install hexo-deployer-git –save
  • 按上述步骤操作后，即可访问你的博客了

美化自己的博客

  • 可以百度搜索hexo主题，找到git地址后，直接在hexo文件夹下的themes文件夹下 git base here 来clone主题

  • 修改hexo文件夹下的配置文件_config.yml，将theme参数为对应clone的主题

  • 部署主题，本地查看效果，执行完一下命令后在浏览器输入localhost:4000

  • 如果对主题满意，那么可以将其部署到github
    打开hexo文件夹，右键git base here

打开个人主页，即可看到修改后的效果

撰写自己的第一篇博客

  • 打开hexo文件夹，右键 git base here，输入 hexo n “你的标题”，对应在source/_posts文件夹下生成与标题对应的.md文件
  • 在此文件中书写个人博客
  • 写完之后在hexo文件夹下 git base here，输入命令

Host github.com
User git
Hostname ssh.github.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/id_rsa
Port 443
